1. **Navigiere ins Backend-Verzeichnis:**

      ```bash
    node server.js

   ```bash
   cd frontend
5. **Abhängigkeiten installieren**
   
   ```bash
   npm install
   
6.. **Starte Frontend**
   ```bash
    vite / npm run dev
